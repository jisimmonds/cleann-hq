# CLEANN HQ

Internal marketing ticketing portal — clickable prototype.

Live demo: https://jisimmonds.github.io/cleann-hq/

Built as an HTML prototype to validate the UX before building the production version. Open index.html in a browser to view locally.

## What's in here

- Asset library with type-aware previews (labels, decks, PDFs, photos, video)
- Ticket submission flow with deadline + estimated completion date distinction
- At-Risk auto-flagging when ECD is later than the deadline
- Branded with CLEANN's primary palette and typography
