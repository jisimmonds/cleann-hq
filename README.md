# CLEANN HQ

Internal information hub + marketing ticketing portal for CLEANN.

Live demo: https://jisimmonds.github.io/cleann-hq/

A clickable prototype validating the UX before production build. Open index.html in any browser to view locally.

## What's in here

- Daily information hub — Wolcott weather, hemp beverage industry headlines, at-a-glance ticket pipeline
- Asset library — type-aware previews (labels, decks, PDFs, photos, video) with preview file + source file pattern
- Ticket system — sales sets the deadline, marketing sets the estimated completion date, system auto-flags "At Risk" when ECD is later than the deadline
- Brand-native UI — built on CLEANN's palette (CLEANN Blue, Cider Amber, Soft Cyan) and typography (IvyPresto serif + Proxima Nova sans)
